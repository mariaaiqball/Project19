# Project 19 — Consulting Website

A professional single-page website for Project 19, a technology consulting agency serving small and medium local businesses.

## Quick Start

Open `index.html` in your browser, or serve locally:

```bash
# Python
python3 -m http.server 8080

# Node (if npx is available)
npx serve .
```

Then visit [http://localhost:8080](http://localhost:8080).

## Sections

- **Hero** — Value proposition with consultation CTA
- **About** — Mission and pain points addressed (budget, expertise, delivery)
- **Values** — Results, Honesty, Reliability
- **Approach** — 4-step engagement process
- **Case Studies** — Three client stories with measurable outcomes
- **Contact CTA** — Free consultation request modal

## Customization

- Update case study content in `index.html`
- Change contact email in the footer (`hello@project19.io`)
- Connect the consultation form in `script.js` to your backend or email service
- Logo lives at `public/assets/logo.png`
